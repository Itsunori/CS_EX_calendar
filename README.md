## 使い方
clone
```
git clone --recursive git@github.com:Itsunori/CS_EX_calendar.git
```
実行(db)
```
cd backend/com/db/jajujoDBcd 
javac jajujoDB/JajujoDB.java
java jajujoDB/JajujoDB
```

実行(backend)
```
cd backend
javac com/backend/Backend.java
java com/backend/Backend
```

実行(frontend)
```
cd frontend
javac Index.java
java Index
```
